# Sistemi Operativi 2 - Appunti

Benvenuto negli appunti del corso di Sistemi Operativi 2. Questo sito raccoglie i concetti fondamentali del modulo in una forma organizzata per lezioni, con un focus sull'intuizione logica dietro ogni argomento e sui dettagli tecnici necessari per lo studio.

## Come usare questi appunti

- Segui l'ordine delle lezioni nella sidebar per un percorso lineare.
- Ogni capitolo contiene una sezione di "Intuizione Logica" e una lista di "Punti Chiave".
- Le note sono pensate per essere riusate nella stesura di un sito di studio.
