Variabili statiche
• Possono essere locali o esterne
• Locali:
• dichiarate all’interno di una funzione
• sono mantenute tra una chiamata e l’altra della funzione
• Costituiscono una memoria privata della funzione

Le variabili static vengono
automaticamente inizializzate (una volta)

Variabili statiche
void fun() {
static int myvar;

//inizializzata automaticamente a 0

myvar++;
printf("myvar = %d\n", myvar);
}
void main(void) {
fun();

Il codice stampa:

fun();

myvar = 1
myvar = 2

}

Variabili statiche esterne
• Sono variabili statiche dichiarate al di fuori di ogni funzione (in un
dato file sorgente)
• Sono visibili dal punto di dichiarazione fino alla fine del file sorgente

Funzioni statiche
• Non visibili al di fuori del file sorgente in cui sono dichiarate
• La clausola static permette quindi di variare la visibilità di un
oggetto, non solo la sua permanenza in memoria

Funzioni statiche
// file fstatic.c

// file fun.c

void fun() {

Ricorsione
• In C è possibile scrivere funzioni ricorsive, cioè che chiamano se
stesse
• Svantaggi: in generale codice più lento e più consumo di memoria
(perché?)
• [Variabili locali/automatiche salvate nello stack.]

• Vantaggi: codice più compatto e semplice da scrivere
• Esercizio [max 5 minuti!]: scrivere una funzione ricorsiva per il
fattoriale di un intero (provatela con numeri minori di 20…)

Puntatori a funzione
• Il C permette puntatori a funzione, anche se una funzione NON è una
variabile!
• Dichiarazione:
int (* funptr)();
//funptr è un puntatore ad
una funzione che ritorna
un intero
• Chiamata:
int a;
a = (* funptr)();
